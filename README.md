# ngs.io

Personal blog by Atsushi Nagase

https://ngs.io/

## Tech Stack

- [Hugo](https://gohugo.io/) - Static site generator
- [hugo-primer-blog](https://go.ngs.io/hugo-primer-blog) - Theme (Hugo Module)
- GitHub Pages - Hosting

## Local Development

```bash
# Start dev server
hugo server -D

# Build
hugo --gc --minify
```
